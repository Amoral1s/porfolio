Основная специализация — проектирование сайов и landing page.

Какими языками владею и пользуюсь?

  • html + css - разметка bootstrap4 для удобного позиционирования обьектов и адаптации под устройства с любыми размерами экранов.
  • JavaScript - препроцессором jQuery, который многократно сокращает создание вашего проекта, никак не сказываясь на скорости загрузки и качестве.
  • PhP - для создания отзывчивости сайта, создание форм отправок email.
  • CSM система WordPress - очень популярная система управления вашим сайтом, позволяет менять вам контент не залезая в код.
  • gulp+sass - позволяет в разы ускорить создания проекта и очистки от ненужного мусора на стадии передачи его вам.
  • Мои знания с каждым днем увеличиваются, в дальейшем этот список расширится...
